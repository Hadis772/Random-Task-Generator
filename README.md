# Random Task Generator – Генератор случайных задач

**Автор:** Хадис Кулиев

## Описание
Приложение с графическим интерфейсом (Tkinter) для генерации случайных задач из заранее заданного или пользовательского списка.  
Основные возможности:
- Генерация случайной задачи (кнопка **Сгенерировать задачу**).
- Отображение **истории** всех сгенерированных задач с временной меткой.
- **Фильтрация** истории по типу задачи (учеба / спорт / работа).
- **Добавление** новых задач в общий пул (с проверкой на пустое название).
- Автоматическое **сохранение** истории и списка задач в JSON‑файлы (`history.json`, `tasks.json`).
- Данные загружаются при запуске – после перезапуска приложения ваши задачи и история не теряются.

## Требования
- Python 3.6+
- Стандартные библиотеки: `tkinter`, `random`, `json`, `os`, `datetime`
